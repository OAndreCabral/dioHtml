Implementa uma pagina de html com explicações sobre as tags com link

  • h1,h2,h3,h4,h5
  • p
  • mark
  • small
  • i
  • u
  • strong
  • ol
  • ul
  • li
  • a
  • hr
  • sub
  • sup
  • blockquote

  1. As tags H referece aos titulos são usadas para criar cabeçalhos, o niveis de h são dados de acordo com a importancia, h1 é a mais importante e h6 a menos importante

  2. a tag p é usada para definir parágrafos de texto

  3. a tag mark é usada para destacar a parte de baixo do texto, ela grifa a parte desejada do texto

  4. Tag small é utilizada para definir texto em um tamanho menor do que o texto que os demais texto do bloco que ele pertence.

  5. Tag i coloca o texto em itálico

  6. Tag u serve para sublinhar o texto

  7. Tag strong deixa o texto em negrito.

  8. Tag ol cria uma lista ordenada ( item 1, item 2, item 3)

  9. Tag ul cria um lista sem order definida

  10. Tag li é usada dentro das tags ol e/ou ul para definir cada item da lista

  11. Tag a serve para criar links

  12. tag hr é utilizada para criar uma linha na horizontal

  13. Tag sub é usada para colocar texto em uma posição subscrita.

  14. Tag sup é usada para colocar o texto numa posição sobreescrita, quase igual a tag sub.

  15. Tag blockquote é usada para citar blocos de texto de outra fonte, quando pretende citar um autor.
